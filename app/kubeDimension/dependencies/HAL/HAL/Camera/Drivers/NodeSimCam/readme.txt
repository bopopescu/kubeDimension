How to build Node2Cam (please build it with GCC 4.7 or higher version, or you may have strange error!)

1, Turn Node and Node2Cam option on when build hal.
2, make -j, you will see an error about Node2Cam.
3, type make -j again and that error will gone. Done. (If you know how to solve this error, please inform luma@gwu.edu)
